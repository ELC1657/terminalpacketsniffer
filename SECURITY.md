# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.12    | ✅ Current |

## Reporting a Vulnerability

If you find a security issue in this project, please open a [GitHub Issue](https://github.com/ELC1657/terminalpacketsniffer/issues) describing the problem.

## Disclaimer

This tool is intended for **educational and authorized network monitoring only**. Only run it on networks you own or have explicit permission to monitor. Unauthorized packet capture may be illegal in your jurisdiction.
